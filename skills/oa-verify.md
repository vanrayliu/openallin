# /oa-verify — 多维度验证

当用户输入 `/oa-verify <phase>` 时执行此技能。

## 工作流程

### 1. 代码质量检查
- 运行 lint
- 运行 typecheck
- 检查代码风格

### 2. 测试验证
- 运行单元测试
- 运行集成测试
- 检查测试覆盖率

### 3. 构建验证
- 确保项目可以构建
- 检查构建产物

### 4. 功能验证
- 验证实现符合规格
- 检查验收标准

### 5. 输出报告
```markdown
# 验证报告

## ✅ 通过
- 检查项 1
- 检查项 2

## ❌ 失败
- 检查项 3: 问题描述
```

## 输出

- 验证报告
- 失败项及修复建议

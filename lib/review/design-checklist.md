# UI/UX Design Review Checklist

> Comprehensive checklist for reviewing UI/UX design consistency and quality.

## UI Consistency

### Color Palette
- [ ] Primary color used consistently
- [ ] Secondary color used for accents
- [ ] CTA color distinct from other colors
- [ ] Background color consistent across pages
- [ ] Text color has sufficient contrast

### Typography
- [ ] Font family consistent (max 2 families)
- [ ] Font sizes follow scale (e.g., 12, 14, 16, 18, 24, 32)
- [ ] Line heights consistent (1.4-1.6 for body)
- [ ] Font weights limited (regular, medium, bold)

### Spacing
- [ ] Margins follow 4px/8px base
- [ ] Padding consistent for similar components
- [ ] Gaps between elements consistent
- [ ] Whitespace sufficient for readability

### Components
- [ ] Buttons have consistent styling
- [ ] Inputs have consistent styling
- [ ] Cards have consistent styling
- [ ] Icons consistent in size and style

## UX Best Practices

### Navigation
- [ ] Clear navigation hierarchy
- [ ] Breadcrumbs for deep pages
- [ ] Back button always available
- [ ] Active state visible

### Forms
- [ ] Labels above inputs
- [ ] Clear validation messages
- [ ] Submit button prominent
- [ ] Auto-focus on first input

### Feedback
- [ ] Loading states for async operations
- [ ] Success messages after actions
- [ ] Error messages near the problem
- [ ] Confirmation before destructive actions

## Accessibility (WCAG 2.1)

### Color Contrast
- [ ] Normal text: >= 4.5:1
- [ ] Large text: >= 3:1
- [ ] UI components: >= 3:1

### Focus States
- [ ] Visible focus outline
- [ ] Focus order logical
- [ ] Focus not trapped

### Screen Reader
- [ ] All images have alt text
- [ ] All inputs have labels
- [ ] ARIA attributes correct
- [ ] Semantic HTML used

## Responsive Design

### Breakpoints
- [ ] Desktop: >= 1024px
- [ ] Tablet: 768px - 1023px
- [ ] Mobile: < 768px

### Touch Targets
- [ ] Min 44x44px for touch
- [ ] Adequate spacing between targets
- [ ] No tiny buttons on mobile

## Visual Polish

- [ ] No layout overflow
- [ ] No horizontal scroll on mobile
- [ ] Smooth transitions (150-300ms)
- [ ] Consistent border radius
- [ ] Consistent shadow styles
